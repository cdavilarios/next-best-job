How it works

Four-stage agent pipeline

Each stage is a distinct module. You can see all the code in the repo. Your data stays private.

01 — Scanner
Pull from job portals
Uses Indeed and Dice APIs via MCP (Model Context Protocol) to search across role types simultaneously. Runs keyword combinations for TPM, AI PM, Enterprise AI, and Strategic Projects roles.
Indeed MCP Dice MCP MCP tool calls
02 — Scorer
Weighted criteria engine
Every role is scored 0–100. Must-haves disqualify. Nice-to-haves add points. Criteria are in src/criteria.js — edit them to match your target profile.
Must-haves Nice-to-haves src/scorer.js
03 — Materials Engine
Tailored application materials
Calls Claude API to generate a tailored cover letter and resume bullet suggestions per role. Two voices: enterprise delivery (big tech, finance) or founder-operator (startups). Resume track auto-selected by role type.
Claude API Enterprise voice Founder voice
04 — Tracker
Pipeline to offer
Kanban board: Applied → Screening → Interview → Offer. Encrypted in localStorage. No backend. No database. Your applications are yours alone.
Encrypted local src/store.js
src/scorer.js — core scoring logic 
// Every role starts at 0. Must-haves disqualify. Nice-to-haves add points.

function scoreJob(job, criteria) {
  // 1. Check must-haves — fail fast
  const salary = extractSalary(job.salary);
  if (salary < criteria.mustHave.minSalaryUSD) {
    return { score: 0, disqualified: true };
  }

  // 2. Base score for passing all must-haves
  let score = 45;

  // 3. Nice-to-haves add up to 55 more points
  for (const nic of criteria.niceToHave) {
    if (matches(job, nic)) score += nic.points;
  }

  // 4. Detect cover letter voice + resume track
  return {
    score: Math.min(100, score),
    voice: detectVoice(job),
    resumeTrack: detectResumeTrack(job),
  };
}

Run your own instance

Fork and deploy in 5 minutes

Fork this repo
Click Fork on GitHub. Name it anything. You get your own copy of the full agent.
Set your password hash
In index.html, find PASSWORD_HASH and replace it with the SHA-256 of your password.
Generate it: echo -n "yourpassword" | sha256sum
Edit your criteria
Open src/criteria.js. Set your salary floor, target locations, role types, and nice-to-haves. The scorer uses these weights automatically.
Enable GitHub Pages
Settings → Pages → Deploy from branch → main → / (root) → Save. Live in ~2 minutes.
(Optional) Add your Claude API key
In the private dashboard → Settings, paste your Anthropic API key. This enables live job scanning, scoring, and cover letter generation. Stored only in your browser.
