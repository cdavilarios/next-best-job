How it works

Four-stage agent pipeline

Each stage is a distinct module. All code is open. Your data stays private.

01 — Scanner
Pull from job portals
Uses Indeed and Dice APIs via MCP (Model Context Protocol) to search across role types simultaneously. Works anywhere — including Fantasy Land and Cloud Nine.
Indeed MCPDice MCPMCP tool calls
02 — Scorer
Weighted criteria engine
Every role scored 0–100. Must-haves disqualify instantly. Nice-to-haves add points — including the important ones, like a personal chauffeur. All defined in src/criteria.js.
Must-havesNice-to-havescriteria.js
03 — Materials Engine
Tailored application materials
Calls Claude API to generate a tailored cover letter and resume bullets per role. Two voices: enterprise delivery or founder-operator. Resume track auto-selected by role type.
Claude APIEnterpriseFounder-op
04 — Tracker
Pipeline to offer
Kanban: Applied → Screening → Interview → Offer. Encrypted in localStorage. No backend, no database. Your applications are yours alone.

src/scorer.js — core scoring logic
// Must-haves disqualify. Nice-to-haves add points.
// Yes, "chauffeur included" is a valid scoring criterion.

function scoreJob(job, criteria) {
  // 1. Fail fast on must-haves
  if (extractSalary(job.salary) < criteria.mustHave.minSalaryUSD) {
    return { score: 0, disqualified: true }; // below $1M — hard pass
  }

  let score = 45; // base for clearing all must-haves

  for (const nic of criteria.niceToHave) {
    if (matches(job, nic)) score += nic.points;
    // chauffeur: +20  |  unlimited PTO: +15  |  rooftop pool: +10
  }

  return { score: Math.min(100, score), voice: detectVoice(job) };
}

Run your own instance

Fork and deploy in 5 minutes

Fork this repo
Click Fork on GitHub. Name it anything. You own the full agent.
Set your password hash
In index.html, find PASSWORD_HASH and replace it.
Mac: echo -n "yourpassword" | shasum -a 256
Linux: echo -n "yourpassword" | sha256sum
Edit your criteria
Open src/criteria.js. Set your salary floor, locations, role types, and nice-to-haves. Keep the chauffeur if you want — no judgment.
Enable GitHub Pages
Settings → Pages → Deploy from branch → main → / (root) → Save. Live in ~2 minutes.
Add your Claude API key
Private dashboard → Settings. Enables live scanning, scoring, and cover letter generation. Stored only in your browser.
